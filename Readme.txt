Added PortableMu for Win64

About:
PortableMu is a - as the name states - portable version of the Mu-Editor.
This one has an additional Calliope mini Mode, so that you can programm your Calliope mini with Mu-Editor 
without the need to install anything.

Download the PortableMu-win64 Files, unzip, and put the "portmu" on a UBS-Drive or in any place on your computer.
To start PortableMu simply double click the "Launch Mu.bat"
It will take a while, depending on your USB-Stick or/and your computer.

You then can go to the Mode-Section and select Calliope-Mode.

German
Über:
PortableMu ist eine - wie der Name vermuten lässt - portable Version des Mu-Editors welche 
ohne eine Installation auskommt.
Diese Version hat einen eigens hinzugefügten Calliope mini Modus, damit du deinen Calliope mini 
mit dem Mu-Editor programmieren kannst.

Lade die PortableMu-win64 Datei herunter, entpacke sie und übertrage den "portmu" Ordner 
auf einem USB-Stick oder irgendwo auf deinem Computer.
Um den PortableMu zu starten, doppelklicke einfach auf die enthaltene "Launch Mu.bat"
Das wird einige Zeit dauern, wie lange ist abhängig von deinem USB-Stick und/oder deinem Computer.

Wenn du dann in die Modus-Auswahl gehst, kannst du den Calliope-Modus auswählen.
